Odoo S3 File Upload

Odoo 19 addon for C3 Edition Publishing — direct browser uploads and downloads for project.task attachments on S3-compatible storage (AWS S3 or Cloudflare R2). Odoo orchestrates auth and metadata; file bytes never pass through the server.

What it covers

  • All attachments on project.task (chatter, drag-and-drop, file picker)
  • Multipart direct upload to S3/R2 with progress notifications
  • Large files beyond Odoo's default 128 MB client limit
  • Permission-checked presigned GET downloads
  • Settings for bucket, endpoint, key prefix, TTL, and blocklist
  • Credentials via environment variables only

How to use it

  • Install the module from Apps in Odoo 19
  • Set S3_ACCESS_KEY_ID and S3_SECRET_ACCESS_KEY on the container
  • Configure bucket and endpoint under Settings → S3 File Upload
  • Add CORS on the bucket for your Odoo origin (PUT, GET, HEAD; expose ETag)
  • Attach files on any project task — uploads go straight to storage
